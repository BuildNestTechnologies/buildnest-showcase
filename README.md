# BuildNest Showcase

Premium business showcase website for BuildNest Technologies. Handcrafted in India with a focus on rich aesthetics, performance, and modern engineering.

## 🚀 Tech Stack
- **Frontend**: React 19 + TypeScript
- **Build Tool**: Vite 8 (Rolldown)
- **Styling**: Tailwind CSS v4 (CSS-first architecture)
- **Animations**: Framer Motion 12
- **Icons**: Lucide React 1.11

## 🛠️ Development

### Setup
```bash
cd frontend
npm install
```

### Dev Server
```bash
npm run dev
```

### Production Build
```bash
npm run build
```

## 🏗️ Architecture
The project follows a modular component structure:
- `src/components/ui`: Radix-based UI primitives.
- `src/components/site`: Custom business components.
- `src/pages`: Application views.
- `src/data`: Centralized content management.
